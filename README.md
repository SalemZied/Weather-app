# Weather Project HTML

Ce dossier contient une application web météo en HTML, CSS et JavaScript.

## Contenu
- Interface d’accueil et de navigation
- Pages météo, galerie et contact
- Intégration de cartes et de données météo

## Utilisation
1. Ouvrez le fichier index.html dans votre navigateur.
2. Si nécessaire, utilisez un serveur local pour éviter certains problèmes de chargement.

## Objectif
Ce projet présente une interface météo interactive et visuelle.
